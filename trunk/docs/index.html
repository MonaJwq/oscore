OSCore is a set of core utility-classes, frameworks and modules that are common to the other components of OpenSymphony. Stand- alone these components are not complete systems, but they are essential building blocks for J2EE based applications. OSCore consists of some library classes which aid in common functions, and some additional specialized modules.

  • Requirements - OSCore requires Jakarta commons-logging. If you are using the SequenceGenerator EJBs, you also will need an EJB 1.1 or higher compliant container as well.
